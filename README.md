O que é preciso em um jogo de TicTacToe?

- 2 jogadores
- 1 tabuleiro 3x3

Condição de vitória?
R: 3 peças do mesmo jogador em uma linha reta, independente da direção

Condição de empate?
R: Todos campos preenchidos e nenhuma linha contínua de peças

-- X --

Controlador do tabuleiro
-Criar tabuleiro
-Printar tabuleiro
-Posicionar peças no tabuleiro

Controlador do jogo
-Criar os jogadores
-Permitir jogada
-Checar estado de vítoria ou empate
-Trocar turno
